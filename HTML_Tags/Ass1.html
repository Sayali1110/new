Stack

A stack is a linear data structure that follows the LIFO (Last-In-First-Out) principle. This means that the last element added to the stack is the first one to be removed.Think of it like a stack of plates: you add a new plate to the top, and when you want to take a plate, you take the top one first.

Tree

A tree is a non-linear data structure that represents a hierarchical relationship between nodes. It consists of nodes connected by edges, with a single node at the top known as the root node. Each node can have zero or more child nodes, and each child node can have its own set of children, forming a hierarchical structure
