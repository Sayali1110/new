Exception Handling

Exception handling is a mechanism in programming languages to manage and respond to errors or unexpected events that occur during program execution.

Try-Catch Block: A code block that encloses the code that might throw an exception.
Catch Block: A code block that handles a specific type of exception. It is executed when an exception of that type is thrown within the try block.
Finally Block: A code block that is always executed, regardless of whether an exception is thrown or caught. It's often used for cleanup tasks like closing files or releasing resources.
Throw Keyword: Used to explicitly throw an exception.
Throws Keyword: Used to declare that a method might throw a specific type of exception.
